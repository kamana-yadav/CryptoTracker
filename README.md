# 🪙 CryptoTracker - Java

Real-time Cryptocurrency Tracker built with Java.

## Features
- Real-time crypto price fetching via CoinGecko API
- Portfolio management (Add/Remove coins)
- Price alert system
- Total portfolio value calculation

## Tech Stack
- Java
- CoinGecko REST API
- OOP Concepts
- ArrayList Collections

## Files
- CryptoData.java — Coin data model
- CryptoAPI.java — API integration
- PortfolioManager.java — Portfolio logic
- PriceAlert.java — Alert system
